### 🧱 原版为核心，体验至上
我们致力于打造最纯粹的生存体验。在 **Java 1.21.8** 的基础上，我们仅添加了适度的功能增强，确保在不破坏游戏平衡的前提下，提升多人协作的便利性。
- **传送系统**：方便玩家跨越地图，寻找伙伴。
- **经济商店**：完善的交易体系，让资源流通更高效。

### 👑 尊重主权，数据无忧
在 OurMC，**玩家的劳动成果是神圣不可侵犯的**。
- **管理承诺**：腐竹亲自运维，严禁滥用权限，绝不随意回档或清空数据。
- **公平环境**：透明的社区规则，确保每一份建筑、每一箱物资都受到保护。

### 👥 活跃社区，告别单机

- **包容氛围**：无论你是红石大佬、建筑大师，还是刚入坑的萌新，都能在这里找到同伴。

### 🎲 娱乐拓展，拒绝枯燥

## 📥 加入我们

只需简单三步，即可开启你的方块之旅：

1. **版本准备**：启动 Minecraft Java 版 `1.21.8`（推荐使用 HMCL、PCL2 或官方启动器）。
2. **连接地址**：进入 `多人游戏` -> `添加服务器`。
   - **名称**：`OurMC`
   - **地址**：`mc.our-mc.cn`
3. **开始游玩**：进入服务器后按照提示完成初始化即可。

::: tip 温馨提示
服务器**允许离线登录**。为了保障您的账户安全与数据一致性，请务必使用**唯一的游戏名**，并建议尽快联系管理员进行角色绑定。
:::
