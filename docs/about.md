## 📥 加入方式

1. 启动 **Minecraft Java 版**，版本选择 `1.21.8`（推荐使用官方启动器或 HMCL、PCL2 等）。
2. 点击 **多人游戏** → **添加服务器**：
   - 服务器名称：`OurMC`
   - 服务器地址：`mc.our-mc.cn`
3. 加入服务器，开始你的自由生存之旅。

> 💡 提示：服务器**允许离线模式**，无需正号登录。但请使用**唯一的游戏名**，避免与他人重名造成数据错乱。
