# 什么是 OurMC

OurMC 是一个非商业化的 Minecraft 公益服务器。我们旨在提供一个稳定、自由且内容丰富的生存环境。

## 服务器架构
- **版本**：Fabric 1.21.8（支持原版客户端直接加入）
- **类型**：生存 + 周期性小游戏
- **费用**：完全公益，不设氪金，不接受赞助

## 玩法内容
1. **生存体验**：在原版基础上加入了农夫乐事、工业、更多结构、地形优化及末地增强。
2. **趣味插件**：包含家具、钓鱼增强，以及可以在游戏内玩 MC 的特殊机制。
3. **小游戏**：设有独立的小游戏服，内容会定期更换。
4. **管理风格**：无白名单限制，崇尚自由建筑与生存，不干预玩家正常的社交与竞争。
